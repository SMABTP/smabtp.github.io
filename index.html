Complement Outlook pour ESMA

Démonstration d'un fonction
permettant d'accéder au contenu d'un message

1

Ajouter le complément Outlook

Sélectionnez un message et cliquez sur les 3 points.
Choisissez l'option "Télécharger des complements".
Puis cliquez sur l'option "Mes compléments".
Cliquez sur le lien "Ajoutez un complement personnalisé".
Choisissez l'option "A partir d'une url..." et remplir le champ par : https://smabtp.github.io/manifest.xml

2

Lecture du message

Cliquez sur un message et cliquez sur les 3 points.
Une option eSMA est disponible.
En cliquant, un panneaux latéral s'affiche avec le contenu lu du message

3

Stocker le message dans une base de données

Choisissez un des deux boutons.
Le contenu du message et ses pièces jointes sont stockés dans une base de données serveur (json store).
Retrouver les messages sauvegardés. Essayer maintenant !

Contrôle Office 365 avec les API

Utiliser la richesse des données de Microsoft Graph afin de créer des applications pour les organisations et les clients qui interagissent

Graph Explorer

Microsoft Graph est une API web RESTful qui vous permet d’accéder aux ressources de service Cloud Microsoft. Après avoir enregistré votre application et obtenu des jetons d’authentification pour un utilisateur ou un service, vous pouvez effectuer des requêtes dans l’API Microsoft Graph.
